Exvertixe Setup Guide

✓ Website structure created successfully!
Follow the steps below to complete the setup.
1

Firebase Setup

  1. Go to Firebase Console
  2. Create a new project
  3. Enable Authentication (Email/Password)
  4. Create Firestore Database
  5. Get your Firebase config from Project Settings
  6. Update js/firebase-config.js with your credentials
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT_ID.appspot.com",
  messagingSenderId: "YOUR_SENDER_ID",
  appId: "YOUR_APP_ID"
};
2

Cloudinary Setup

  1. Go to Cloudinary
  2. Create a free account
  3. Get your Cloud Name from dashboard
  4. Create an unsigned upload preset in Settings > Upload
  5. Update js/cloudinary.js with your config
const CLOUDINARY_CONFIG = {
  cloudName: 'YOUR_CLOUD_NAME',
  uploadPreset: 'YOUR_UPLOAD_PRESET'
};
3

Create Admin Account

  1. Sign up through the website normally
  2. Go to Firebase Console > Firestore Database
  3. Find your user document in the users collection
  4. Add field: role with value: admin
  5. Access admin panel at /admin.html
4

Run the Website

Choose one of these methods:

Using Python:

python3 -m http.server 8000
# Visit http://localhost:8000

Using Node.js:

npm install -g http-server
http-server
# Visit http://localhost:8080

Using VS Code:

Install "Live Server" extension and right-click on index.html > "Open with Live Server"

5

Setup Checklist

  • Firebase project created
  • Firebase Authentication enabled
  • Firestore Database created
  • Firebase config updated in code
  • Cloudinary account created
  • Cloudinary config updated in code
  • Admin user created
  • Local server running
  • Website accessible in browser
  • Test signup/login working
  • Test admin panel access
⚠️ Important Security Notes:
• Never commit real Firebase credentials to public repositories
• Set up Firestore security rules (see README.md)
• Use HTTPS in production
• Implement proper validation
